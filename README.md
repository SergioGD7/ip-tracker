# Localizador de IP (IP Tracker)

Una aplicación web moderna y sencilla que permite localizar geográficamente direcciones IP y obtener información detallada sobre ellas.

## Características

* **Autodetección**: Detecta automáticamente tu IP pública al cargar la página.

* **Búsqueda Manual**: Permite introducir cualquier dirección IPv4 para obtener sus datos.

* **Información Detallada**: Muestra el proveedor (ISP), ASN, organización, ciudad, país, zona horaria y coordenadas.

* **Mapa Interactivo**: Visualiza la ubicación aproximada en un mapa interactivo (OpenStreetMap) usando Leaflet.

* **Diseño Moderno**: Interfaz oscura con efectos "Glassmorphism", totalmente responsiva (móvil y escritorio).

## Tecnologías Utilizadas

* **HTML5 & CSS3**

* **JavaScript (ES6+)**

* **Tailwind CSS** (Estilos utilitarios)

* **Leaflet.js** (Mapas interactivos)

* **API**: ip.guide (Proveedor de datos IP gratuito)

## Nota sobre la Precisión

La geolocalización basada en IP no es exacta como un GPS. La ubicación mostrada generalmente corresponde a la central o nodo principal de tu Proveedor de Servicios de Internet (ISP) en tu región.

## Créditos

* Datos de IP proporcionados por ip.guide.

* Mapas proporcionados por OpenStreetMap.
